# Qoder

## ภาพรวม

โปรเจกต์นี้เป็นที่เก็บข้อมูลขั้นต้นที่มีไฟล์ข้อความภาษาไทย

## ไฟล์ในโปรเจกต์

- `hello.txt` - ไฟล์ข้อความที่มีคำทักทาย
- `world.txt` - ไฟล์ข้อความภาษาไทย
- `AGENTS.md` - คำแนะนำสำหรับ Qoder ในการทำงานกับโปรเจกต์นี้

## การใช้งาน

ขณะนี้โปรเจกต์ยังไม่มีระบบ build หรือ test ที่ตั้งค่าไว้

## ข้อมูลเพิ่มเติม

สำหรับข้อมูลเพิ่มเติมเกี่ยวกับโครงสร้างโปรเจกต์ กรุณาดูที่ไฟล์ `AGENTS.md`
